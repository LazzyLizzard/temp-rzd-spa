Платформа для локальной модульной разработки чего бы то ни было (в частности системы продаж).

Основная идея - что все запросы кроме бандла страницы проксируются на сервер.

Открывается специальный слой (наример 5901) который имеет в своём составе (не считая темы, которая типа снаружи) только три вещи:
- ресурсы (сообщения ИФР)
- див с id=vue-app
- скрипт с src="build-browser"

Установка:
- добавить в хосты 127.0.0.1 pass2.psi.oooinex.ru
- npm i
- npm run devo

Варианты команд:
- npm run devo - запуск и открытие страницы в браузере
- npm run dev - только запуск
- npm run devw - запуск с авторестартом когда правится конфиг вебпака (требует nodemon)
